# Coding convention

- Imports are added automatically so you don't need to import stuff, also it will crash if you include imports within `client/`
- In every file, you can export one thing, and that one thing should have the same name as the file
- don't use `export default`
    - simply because that is how we have done it so far
- ´./lint.sh´ will take care of most of the coding convention
